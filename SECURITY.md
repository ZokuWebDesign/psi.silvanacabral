# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

For valid security issues, please report them directly to Silvana Cabral's development team via email rather than through public issue tracking.

The security team will process your report promptly and respond within 2-3 business days. We take all security concerns seriously and will coordinate a responsible disclosure when necessary.

## Security Commitments

- **Patient Privacy**: All patient data is handled with the highest standard of confidentiality and care.
- **GDPR & LGPD Compliance**: This project respects and complies with international data protection regulations.
- **Regular Updates**: Dependencies are regularly updated to address security vulnerabilities.
- **Secure Communication**: All contact and booking information is transmitted securely.
